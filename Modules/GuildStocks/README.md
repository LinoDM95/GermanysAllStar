# GuildStocks – Modulübersicht

Einfaches Anzeige-Modul für alle bekannten Gildenbank-Bestände.

- `UI/Main.lua`
  - Liest die Scan-Daten aus `Modules/GuildStockPlanner/Data/DB.lua`.
  - Stellt ein eigenständiges Fenster zur Verfügung, das über den Hub gestartet wird.
  - Zeigt sämtliche Items und ihre Mengen in der Gildenbank.
